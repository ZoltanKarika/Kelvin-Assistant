# Architecture Decision Records

Az ADR-ek röviden rögzítik a hosszú távon fontos technikai döntések
kontextusát, eredményét és következményeit.

Állapotok:

- `Proposed`: még nem elfogadott;
- `Accepted`: érvényes döntés;
- `Superseded`: egy újabb ADR felváltotta;
- `Rejected`: megvizsgált, de elvetett irány.
