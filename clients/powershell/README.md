# PowerShell-kliens

A tervezett Windows kliens a `kelvin` terminálparancsot, az interaktív
munkamenetet és a hostoldali eszközvégrehajtást biztosítja.

Biztonsági feladatai:

- engedélyezett munkakönyvtár ellenőrzése;
- strukturált eszközkérések validálása;
- felhasználói jóváhagyás kérése;
- PowerShell- és Git-folyamatok korlátozott indítása;
- fájlváltozások és parancskimenetek naplózása;
- megszakítás és időkorlát kezelése.

A konkrét kliens-technológia a kliens fejlesztési szakaszában kerül
eldöntésre. Lehetséges irány egy Python-alapú terminálkliens vagy egy natív
PowerShell-modul.
